# 開発ストアでアプリを再インストールする方法

## 概要

開発ストアでアプリを再インストールすることで、最新のURL設定が反映され、古いURLのエラーが解消されます。

## 手順

### ステップ1: Shopify Adminにログイン

1. ブラウザで以下にアクセス：
   ```
   https://admin.shopify.com/store/gift-app-test-01
   ```

2. 必要に応じてログイン

### ステップ2: アプリをアンインストール

1. **左側のメニューから「設定」をクリック**
   - 画面左下の「設定」アイコン（歯車マーク）をクリック

2. **「アプリと販売チャネル」を選択**
   - 設定メニューの中から「アプリと販売チャネル」をクリック

3. **「delivery-gift-lite」を探す**
   - アプリの一覧から「delivery-gift-lite」を探します
   - 見つからない場合は、検索ボックスで「delivery-gift-lite」を検索

4. **アプリを削除**
   - 「delivery-gift-lite」の右側にある「削除」または「アンインストール」ボタンをクリック
   - 確認ダイアログが表示されたら、「削除」または「アンインストール」を確認

### ステップ3: アプリを再インストール

#### 方法1: Partnersダッシュボードから（推奨）

1. **Partnersダッシュボードにアクセス**
   ```
   https://partners.shopify.com
   ```

2. **アプリを選択**
   - 左側のメニューから「Apps」をクリック
   - 「delivery-gift-lite」を選択

3. **開発ストアにインストール**
   - 「Test on development store」または「開発ストアでテスト」ボタンをクリック
   - 開発ストア（gift-app-test-01）を選択
   - 「インストール」をクリック

#### 方法2: 直接URLからアクセス

1. **直接アプリのURLにアクセス**
   ```
   https://admin.shopify.com/store/gift-app-test-01/apps/delivery-gift-lite
   ```

2. **インストールを確認**
   - アプリのインストール画面が表示されます
   - 「インストール」または「Install」ボタンをクリック

3. **権限を承認**
   - 必要な権限（write_productsなど）を確認
   - 「インストール」をクリックして承認

### ステップ4: 動作確認

1. **アプリが正常に表示されることを確認**
   - 「Free Plan」バッジが表示される
   - 「Upgrade to PRO」ボタンが表示される
   - エラーが表示されない

2. **URLを確認**
   - ブラウザのアドレスバーで、最新のURL（`forest-cafe-spice-magnificent.trycloudflare.com`）が使用されていることを確認

## トラブルシューティング

### アプリが見つからない

**原因:**
- アプリが既にアンインストールされている
- 別のアカウントでログインしている

**解決方法:**
- ステップ3の方法2（直接URLからアクセス）を試してください

### インストールできない

**原因:**
- 権限の問題
- アプリの設定に問題がある

**解決方法:**
1. Partnersダッシュボードでアプリの設定を確認
2. 最新のバージョン（delivery-gift-lite-6）がActiveになっていることを確認
3. ブラウザのキャッシュをクリアしてから再試行

### まだ古いURLが表示される

**原因:**
- ブラウザのキャッシュが残っている
- セッション情報が古い

**解決方法:**
1. ブラウザのキャッシュを完全にクリア
2. シークレット/プライベートモードで再試行
3. 別のブラウザで試す

## 注意事項

- 再インストール後、アプリのデータは保持されます（セッション情報のみが更新されます）
- 開発環境でのみ実行してください（本番環境には影響しません）
- 再インストール後、最新のURL設定が自動的に使用されます
