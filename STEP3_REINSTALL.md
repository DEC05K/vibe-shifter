# ステップ3: シークレットモードでアプリを再インストール

## 手順

### 1. 新しいシークレット/プライベートウィンドウを開く

- **Chrome/Edge**: `Ctrl+Shift+N` (Mac: `Cmd+Shift+N`)
- **Firefox**: `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`)
- **Safari**: `Cmd+Shift+N`

### 2. 直接アプリのURLにアクセス

シークレットウィンドウで、以下にアクセス：
```
https://admin.shopify.com/store/gift-app-test-01/apps/delivery-gift-lite
```

### 3. インストールを確認

1. アプリのインストール画面が表示されます
2. 「インストール」または「Install」ボタンをクリック
3. 必要な権限（`write_products`など）を確認
4. 「インストール」をクリックして承認

### 4. 動作確認

インストール後、以下を確認してください：

1. **アプリが正常に表示される**
   - 「Free Plan」バッジが表示される
   - 「Upgrade to PRO」ボタンが表示される
   - エラーが表示されない

2. **URLを確認**
   - ブラウザのアドレスバーで、最新のURLが使用されていることを確認
   - 現在の最新URL: `https://forest-cafe-spice-magnificent.trycloudflare.com`

## トラブルシューティング

### インストール画面が表示されない

**原因:**
- URLが間違っている
- アプリが既にインストールされている

**解決方法:**
- URLを再確認してください
- アプリが既にインストールされている場合は、そのまま使用してください

### エラーが表示される

**原因:**
- 権限の問題
- アプリの設定に問題がある

**解決方法:**
- エラーメッセージの内容を確認してください
- 必要に応じて、Partnersダッシュボードでアプリの設定を確認してください
