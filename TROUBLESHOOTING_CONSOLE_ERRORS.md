# コンソールエラーのトラブルシューティングガイド

## 現在のエラー

コンソールに大量の`net::ERR_BLOCKED_BY_CLIENT`エラーが表示されています。これは、ブラウザの拡張機能（広告ブロッカーなど）がShopifyのサービス通信をブロックしている可能性が高いです。

## エラーの詳細

以下のShopifyサービスへのリクエストがブロックされています：
- `error-analytics-sess...n.shopifysvc.com` - エラー分析サービス
- `otlp-http-production...vc.com/v1/metrics` - メトリクス送信サービス
- `monorail-edge.shopif...vc.com/v1/produce` - データ送信サービス

## 解決方法

### ステップ1: ブラウザの拡張機能を無効化（最重要）

1. **Chrome/Edgeの場合:**
   - アドレスバーに `chrome://extensions/` と入力
   - すべての拡張機能を一時的に無効化
   - 特に広告ブロッカー（AdBlock、uBlock Originなど）を確認

2. **Firefoxの場合:**
   - アドレスバーに `about:addons` と入力
   - すべての拡張機能を一時的に無効化

3. **Safariの場合:**
   - Safari > 環境設定 > 拡張機能
   - すべての拡張機能を一時的に無効化

### ステップ2: シークレットモードで試す

拡張機能が無効化された状態で、シークレットモード（プライベートブラウジング）でアプリにアクセス：

```
https://admin.shopify.com/store/gift-app-test-01/apps/delivery-gift-lite
```

### ステップ3: アプリが実際に表示されているか確認

コンソールエラーは表示されていても、アプリ自体は正常に動作している可能性があります。

以下を確認してください：
- アプリのページが表示されているか
- 「Free Plan」バッジと「Upgrade to PRO」ボタンが表示されているか
- エラーページではなく、通常のダッシュボードが表示されているか

### ステップ4: アプリを再インストール（最後の手段）

上記の方法で解決しない場合：

1. **Shopify Adminにログイン**
   ```
   https://admin.shopify.com/store/gift-app-test-01
   ```

2. **アプリをアンインストール**
   - 設定 > アプリと販売チャネル
   - 「delivery-gift-lite」を探す
   - 「削除」または「アンインストール」をクリック

3. **アプリを再インストール**
   - 左側メニューから「アプリ」をクリック
   - 「delivery-gift-lite」を探してクリック
   - 「インストール」をクリック

## エラーがアプリの動作に影響しない場合

これらのエラーは、アナリティクスやメトリクスの送信がブロックされているだけで、アプリの主要機能には影響しない可能性があります。

アプリが正常に表示され、動作している場合は、これらのエラーは無視しても問題ありません。

## 次のステップ

1. まず、ブラウザの拡張機能を無効化して試してください
2. シークレットモードで試してください
3. アプリが実際に表示されているか確認してください
4. それでも問題が解決しない場合は、アプリを再インストールしてください
