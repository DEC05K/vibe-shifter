# ステップ2: 開発ストアでアプリをアンインストール

## 手順

### 1. Shopify Adminにログイン

以下のURLにアクセス：
```
https://admin.shopify.com/store/gift-app-test-01
```

### 2. アプリと販売チャネルに移動

1. **左側のメニューから「設定」をクリック**
   - 画面左下の「設定」アイコン（歯車マーク）をクリック

2. **「アプリと販売チャネル」を選択**
   - 設定メニューの中から「アプリと販売チャネル」をクリック

### 3. delivery-gift-liteを探す

- アプリの一覧から「delivery-gift-lite」を探します
- 見つからない場合は、検索ボックスで「delivery-gift-lite」を検索

### 4. アプリを削除

1. 「delivery-gift-lite」の右側にある「削除」または「アンインストール」ボタンをクリック
2. 確認ダイアログが表示されたら、「削除」または「アンインストール」を確認

### 5. 確認

- アプリの一覧から「delivery-gift-lite」が消えていることを確認

## トラブルシューティング

### アプリが見つからない

**原因:**
- 既にアンインストールされている
- 別のアカウントでログインしている

**解決方法:**
- その場合は、ステップ3（再インストール）に直接進んでください

### 削除ボタンが見つからない

**原因:**
- UIが変更されている
- 権限の問題

**解決方法:**
- アプリの詳細ページから削除を試みてください
- または、Partnersダッシュボードからアンインストールを試みてください

## 次のステップ

ステップ2が完了したら、「ステップ2完了」とお知らせください。
その後、ステップ3（シークレットモードでアプリを再インストール）に進みます。
